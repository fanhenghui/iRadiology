Reports Standard Charts

iRadiology v1.0 is designed to generate standard chart diagrams based on specified inputs. The reporting capabilities are limitless and is perhaps one of the strongest feature of this application.

Custom reports and charts can be generated upon request as much as needed for research, educational, statistical purposes etc. These reports would then be part of subsequent versions of this software.
